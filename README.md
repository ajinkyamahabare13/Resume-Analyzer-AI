# Resume Analyzer AI

A Flask-based web application that analyzes PDF resumes and matches them against required job skills.

## Features

* Upload PDF Resume
* Extract Resume Text
* Detect Skills Automatically
* Calculate Match Score
* Show Missing Skills
* Store Analysis History in SQLite

## Technologies Used

* Python
* Flask
* SQLite
* PyPDF2
* Bootstrap

## Run Project

```bash
pip install flask PyPDF2
python app.py
```

Open:

http://127.0.0.1:5000
